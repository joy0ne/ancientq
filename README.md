## Build for Android (Google Play)

### Prerequisites
- Node.js 18+
- Android Studio (latest)
- Java 17+ (JDK)

### Steps

```bash
# 1. Install dependencies
npm install

# 2. Build web + initialize Capacitor + add Android platform
npm run android:init

# 3. Build and sync to Android
npm run android:sync

# 4. Open in Android Studio
npm run android:open
```

In Android Studio:
1. Wait for Gradle sync to complete
2. Connect a device or start an emulator
3. Click **Run** (▶️)

### Generate Signed APK/AAB for Play Store

In Android Studio:
1. **Build** → **Generate Signed Bundle / APK**
2. Choose **Android App Bundle (.aab)** for Play Store
3. Create or select a keystore (KEEP THIS SAFE — you need it for every update)
4. Build release variant

### Play Store Upload

1. Go to [Google Play Console](https://play.google.com/console)
2. Create new app → fill details
3. Upload the .aab file
4. Set pricing: **Free** (no in-app purchases, no ads)
5. Fill content rating questionnaire
6. Submit for review (~1-3 days)
